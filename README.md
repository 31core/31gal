# 31gal

## Introduction

A galgame engine written in Rust with SDL2 backend.

## Build

Install SDL2 development libraries

On Debian/Ubuntu:

```shell
sudo apt install libsdl2-dev libsdl2-ttf-dev libsdl2-image-dev
```

On macOS:

```shell
brew install sdl2_ttf sdl2_image
```

Build 31gal

```shell
cargo build --release
```
